# 🏥 Hospital Management System

A Java-based application designed to manage hospital operations efficiently, including patient records and basic hospital workflows.

---

## 🌟 Features

- 👨‍⚕️ **Add Patient**  
  Register new patients with details.

- 📋 **View Patients**  
  Display all patient records.

- ✏️ **Update Patient Details**  
  Modify patient information.

- ❌ **Delete Patient**  
  Remove patient records.

---

## 💻 Tech Stack

- Java  
- JDBC   
- MySQL 
- Maven  

---
